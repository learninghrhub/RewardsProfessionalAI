# RewardAI Professional

RewardAI Professional is a Streamlit compensation review platform for salary review, pay positioning, market benchmarking, merit planning, promotion analysis, lump-sum planning, budget scenarios, risk governance, and executive reports.

## Files in this GitHub-ready version

Upload only these files to your GitHub repository root:

- `app.py`
- `requirements.txt`
- `README.md`

No folders are required. The app generates sample data and blank/sample workbooks internally.

## Run locally

```bash
python -m pip install -r requirements.txt
python -m streamlit run app.py
```

## Deploy on Streamlit Community Cloud

Use these settings:

- Repository: your GitHub repository
- Branch: `main`
- Main file path: `app.py`

## Important data note

Use anonymized data for demos. Compensation data is confidential and should be handled according to company HR, Finance, Legal, and privacy controls.
